    ❓ Help

    What do the stages of the limit map mean?

    1: fast probe search brackets the accel limit · 2: reversal-stress validation · 3: a simulated print at full current is the final judge · 4: the lowest motor current that still holds the point (cooler motor), confirmed with one more long run.

    "Endstop … incorrect phase" error?

    Remove [endstop_phase] (and any [endstop_phase stepper_*]) from printer.cfg and restart — the plugin detects lost steps without it, and that module aborts homing exactly when a step is lost.

    Results differ a lot between runs?

    Motor temperature matters. Let the motor cool between runs and compare runs made under the same conditions. Also make sure belts and grub screws are tight.

    Where are my results?

    Every run saves a CSV and an interactive HTML report into the Speedtest folder inside your config directory. Open the HTML in a browser.

    Testbench mode?

    For a single motor on a bench (only X wired): set testbench: True in [speed_test] or add TESTBENCH=1. Only X is homed and checked.
